# Animated Icons Directory

This directory should contain your animated icons (GIFs or MP4 videos) related to your research areas.

## Required Icons

- `brain.gif` or `brain.mp4` - Brain/Neuroimaging icon
- `dna.gif` or `dna.mp4` - DNA/Genomics icon
- `ai.gif` or `ai.mp4` - AI/Deep Learning icon
- `llm.gif` or `llm.mp4` - Large Language Model icon

### Programming Languages

- `java.gif` or `java.mp4` - Java icon
- `go.gif` or `go.mp4` - Go/Golang icon

### ML Frameworks

## Recommendations

1. **File Format**: MP4 videos are preferred over GIFs for better quality and smaller file sizes
2. **Size**: Keep icons small (64x64 to 128x128 pixels recommended)
3. **Duration**: Short loops (2-5 seconds) work best
4. **File Size**: Try to keep each file under 500KB for fast loading

## Where to Get Icons

You can find animated icons from:
- [LottieFiles](https://lottiefiles.com/) - Convert to MP4
- [IconScout](https://iconscout.com/) - Animated icons
- [Giphy](https://giphy.com/) - GIFs (convert to MP4 for better quality)
- Create custom animations using tools like After Effects, Blender, or online tools

## Conversion Tools

- Convert GIF to MP4: Use FFmpeg or online converters
- Optimize MP4: Use HandBrake or similar tools
- Create from scratch: Use animation software or AI tools

## Note

If you don't have icons yet, the website will still work but the icons won't display. You can add them later by placing files in this directory with the names listed above.
