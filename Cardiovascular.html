Intelligent Solution for Early Stage Prediction of Cardiovascular Diseases Using Artificial Neural Network and Multi-Agent Systems

Thesis Abstract

Cardiovascular disease is one of the leading causes of death around the world and remains as a major public health problem. Apart from that, early-stage cardiovascular disease diagnosis and cardiovascular disease prediction also signifies a major concern in public health. Most of the time, misdiagnosis of cardiovascular diseases is very common occurrence through the world, for cardio specialists. This platform consists with main three main modules, which are web dashboard application, machine learning module and machine learning web service modules. Machine learning module is based on Artificial Neural network and that was designed using TensorFlow Machine Learning framework. Web dashboard was designed using Flask framework and that allows the end user to input clinical variables for diagnose the process for predict the cardiovascular risk stage. The machine learning web service module was based on REST web services that handle the request-reply between ML module and frontend dashboard. The evaluation of the proposed machine learning classifier model performance (accuracy, sensitivity, and specificity) in this research study showed that the proposed model predicted the class label correctly (above 73%). More than seventy three percent of respondents (including cardio specialists) who participated in the evaluation session agree till strongly agreed that this research followed medical procedures that the result can support early-stage cardiovascular disease diagnosis analysis.

Research Dominas: Cardiovascular Informatics (Cardioinformatics), Deep Learning, Machine Learning, Multi-Agent Systems, Visualizations
